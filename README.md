# SentinelAC

![SentinelAC Logo](docs/assets/Logo.png)

## Description

SentinelAC is a free and open-source next-generation anticheat plugin for Spigot, Paper, and their forks.  
It provides reliable cheat detection with minimal server impact, designed to keep your Minecraft server fair and enjoyable.

---

## Features

- Advanced cheat detection algorithms  
- Multi-version support (Spigot, Paper, and forks)  
- Lightweight and performant  
- Open-source and community-driven  

---

## Installation

1. Download the latest release from the [Releases](https://github.com/yourusername/SentinelAC/releases) page.  
2. Place the `.jar` file into your server's `plugins` folder.  
3. Restart or reload your server.  
4. Configure the plugin by editing the config files in `plugins/SentinelAC/`.

---

## Usage

- Once installed, the anticheat will start monitoring players automatically.  
- Customize detection settings in the config file to suit your server needs.  
- Use commands like `/sentinel reload` to reload the configuration without restarting.
